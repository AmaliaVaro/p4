En esta práctica se ha creado una página web de productos de cosmética. Consta de 4 páginas interconectadas:
1. index.html es la página principal desde la cual puedes registrarte, ver qué tienes en el carrito o meterte en el catálogo de cosméticos.
2. cosmeticos.html donde se pueden observar distintos productos. Además, al final se encuentran los enlaces a distintas marcas de maquillaje.
3. login.html te permite inicar sesión o registrate.
4. micarrito.html se encuentran los productos que se han ido añadiendo al carrito, el precio total, las unidades de cada producto y si quieres pagar.
